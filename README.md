# COVID-19-Data-Analysis-using-Python
Python-based project for analyzing and visualizing COVID-19 data using REST APIs, Pandas, NumPy, SQLite, and Matplotlib with fallback support for offline execution.

 Project Overview
This project performs COVID-19 data analysis using Python by fetching real-time data from a public API.
It processes, stores, analyzes, and visualizes the data using popular Python libraries.

Technologies Used
- Python
- Requests (API handling)
- Pandas & NumPy (data processing)
- SQLite (database storage)
- Matplotlib (visualization)

 Features
- Fetches live COVID-19 data using REST API
- Uses fallback data for offline execution
- Stores data in SQLite database
- Performs statistical analysis
- Visualizes top affected states using bar charts

 How to Run:
python -m pip install requests pandas numpy matplotlib
python covid_data_analysis.py
